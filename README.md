# Pro ATS & Modern Resume Builder

A professional, feature-rich web-based resume builder designed to create ATS-friendly and modern resumes with ease. Developed by Aswin.

## Features

- **Multiple Templates**: Choose from Classic ATS, ATS Modern Split, ATS Centered, and Modern with Photo.
- **Rich Text Editor**: A WYSIWYG editor with a ribbon toolbar offering formatting options (fonts, sizes, bold, italic, colors, alignment, lists, line spacing).
- **PDF Export**: Generate high-quality PDFs using `html2pdf.js`.
- **Text Extraction**: Upload an old PDF resume to extract text using `pdf.js` for easy data migration.
- **Customization Options**: Adjust theme colors, upload profile pictures, and customize layout page margins.
- **Interactive Forms**: Easily add, edit, or rearrange Work Experience, Education, Projects, and Skills.

## Technologies Used

- **HTML5 & CSS3**: For structure, responsive styling, and UI animations.
- **JavaScript (Vanilla)**: For dynamic form handling, data binding, text manipulation, and PDF generation.
- **Libraries**:
  - `pdf.js` (PDF text extraction)
  - `html2pdf.js` (PDF generation)
  - `Sortable` (Drag-and-drop functionality)
  - Google Fonts (`Inter`)

## How to Use

1. **Open the App**: Simply open `index.html` in your web browser.
2. **Fill Details**: Enter your personal information, work experience, education, skills, and summary using the left-hand form panel. (You can also upload an old PDF resume to extract text!).
3. **Select Template**: Use the top menu to select your desired resume template.
4. **Edit & Format**: Click directly on the resume preview to make rich-text edits, format text, insert links or images, and change layouts.
5. **Download PDF**: Enter your preferred file name and click the "Download PDF" button to export your completed resume.
